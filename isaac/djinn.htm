Missed Djinn

If you missed out on getting some Djinni in the first game or if you didn't use a password, then you have a chance to get one Djinni of each element in this game.

Venus - Treasure Isle
Mars - Magma Rock
Jupiter - SW Atteka Islet
Mercury - Prox
